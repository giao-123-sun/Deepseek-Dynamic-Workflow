# DDW v0.1.0-alpha Release Notes

DDW v0.1.0-alpha is a non-commercial source-available release candidate for
cache-first dynamic workflows on Open Dynamic Workflows, DeepSeek, Native DDW
workers, and ReasoniX harness agents.

## Highlights

- ODW custom adapter configs for Native, ReasoniX, and mixed backend workflows.
- DeepSeek usage ledger with prompt cache hit/miss aggregation.
- Repomix stable prefix builder for cache-first workflows.
- ReasoniX harness wrapper with transcript parsing and artifact manifest output.
- Workflow dashboard with phases, agent dots, cache bars, backend chips, and
  expandable text artifact previews.
- Five release demos covering cache ROI, codebase audit, policy conflict
  mining, multi-city research, and CDP-ready browser evidence protocols.
- Structured demo handoff format: `cf-dw.structured-handoff.v1`.
- Local release gates: `npm run release:audit` and `npm run release:pack`.
- GitHub workflows for CI and tag-triggered source archive artifacts.

## Known Limitations

- The Web/CDP demo defines a browser evidence protocol but does not yet execute
  live browser/CDP actions.
- Cross-phase synthesis currently uses compact structured handoff; direct
  downstream consumption of `artifact-manifest.json` is planned next.
- Dashboard artifact previews support text-like artifacts; image and screenshot
  thumbnails are planned next.
- Live DeepSeek demo audit is intentionally manual because it requires a
  private `DeepSeek_API_KEY` and incurs API cost.
